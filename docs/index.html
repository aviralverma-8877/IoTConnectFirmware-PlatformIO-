How it works

  1. Plug your ESP8266 board into a USB port. Make sure no other serial terminal is open.
  2. Select your board and install type above, then click Install IoTConnect.
  3. Choose the correct COM/serial port from the browser dialog.
  4. Full Install will erase the device and flash both the firmware and the web UI filesystem. Your previous config will be lost.
  5. Firmware Only flashes only the firmware binary — your Wi-Fi credentials and device settings are preserved.
  6. After flashing, the device reboots. On first install it starts in AP mode (IoTConnect-XXXXXX). Connect to it and open 192.168.4.1 to configure Wi-Fi and MQTT.
